# Generated Go types — canonical record

Auto-generated by `buf generate` from `proto/`. **Do not edit by hand.** This directory is the
canonical generated record of the Go types.

**To produce/consume Kafka messages, use the SDK — not this package.** `sdk/go`
(`github.com/vinirmbrozz/protobuf-contracts/sdk/go`) bundles these generated types **plus** the
Confluent Schema Registry serde (`Bind` / `Produce` / `Consume`). See the root
[README](../../README.md) and the [wire spec](../../docs/confluent-sr-serde-spec.md).

## Adding a contract

Edit `proto/`, then `buf generate` regenerates this directory (CI does it automatically in
`.github/workflows/generate.yml`). Commit `proto/` and the generated output together.
